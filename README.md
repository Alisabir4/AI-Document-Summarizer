Project Overview

AI Document Summarizer is an intelligent document processing workflow built with n8n that automatically extracts text from PDF files, generates concise AI-powered summaries, enriches information using Wikipedia, and stores the results in Google Sheets for easy tracking and management.

Features
Upload PDF documents through a web form.
Automatically extract text from uploaded files.
Generate accurate summaries using Google Gemini 2.5 Flash.
Enhance summaries with additional context from Wikipedia.
Store document titles, summaries, and submission dates in Google Sheets.
Fully automated workflow with no manual processing required.
Tools & Technologies Used

Workflow
User uploads a PDF document through a form.
The workflow extracts text from the PDF.
Google Gemini analyzes and summarizes the content.
Wikipedia provides additional contextual information when needed.
The generated summary is automatically saved to Google Sheets along with document details.
