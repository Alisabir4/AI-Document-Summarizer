# 📄 AI Document Summarizer

An AI-powered document summarization workflow built with **n8n**, **Google Gemini 2.5 Flash**, and **Google Sheets**. This project automatically extracts text from PDF documents, generates concise summaries using AI, and stores the results for future reference.

## 🚀 Features

* Upload PDF documents through a simple form.
* Extract text automatically from uploaded files.
* Generate AI-powered summaries using Google Gemini 2.5 Flash.
* Enhance understanding with Wikipedia integration.
* Store document titles, summaries, and submission dates in Google Sheets.
* Fully automated end-to-end workflow.

## 🛠️ Technologies Used

* n8n
* Google Gemini 2.5 Flash
* Google Sheets
* PDF Text Extraction
* Wikipedia Tool
* Workflow Automation
* JSON Configuration

## 📋 Workflow

1. User uploads a PDF document.
2. Text is extracted from the document.
3. Google Gemini analyzes and summarizes the content.
4. Wikipedia provides additional contextual information when relevant.
5. Summary and document details are automatically saved to Google Sheets.

## ⚙️ Setup

1. Import the workflow JSON file into n8n.
2. Configure Google Gemini API credentials.
3. Configure Google Sheets credentials.
4. Activate the workflow.
5. Upload a PDF document and receive an AI-generated summary.
